A fork of the legendary Nokia mobile phone game designed for the PicoPad device. 
The author is a great programmer KEVINARCZ - https://github.com/NetCzech.
  
For the actual compilation, clone the great PicoLibSDK from here https://github.com/Panda381/PicoLibSDK (tested version 1.04) and copy the PicoLibSDK directory structure into it.
For more information with the compilation follow https://github.com/Panda381/PicoLibSDK#readme
For immediate use the SNAKE.UF2 file can be loaded directly into PicoPad or opened with a pre-loaded loader from the SD card.
